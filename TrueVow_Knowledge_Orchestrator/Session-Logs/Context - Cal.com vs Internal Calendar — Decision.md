# Cal.com vs Internal Calendar — Decision

Customer Portal already has a built-in calendar (app/api/calendar endpoint, dashboard calendar components, analytics event tracking). This internal calendar captures copies of calendar events into the portal. Cal.com (39.7k stars) is a SCHEDULING/BOOKING tool (like Calendly) for external appointment booking — different use case from an internal event calendar. Verdict: KEEP internal calendar in Customer Portal for event management. ADD Cal.com only if law firms need public-facing appointment booking pages for client consultations. Currently, don't add.
